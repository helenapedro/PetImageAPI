# Pet Image API

API for fetching random images of dogs and cats.

## Overview

Pet Image API is a simple Node.js Express application that 
provides endpoints to retrieve random images of dogs and cats 
from external APIs.It prevents showing the same image within the same session.


### Prerequisites

- Node.js installed

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/your-repository.git

    Install dependencies:
          npm install
2. Usage
     Start the server:
          npm start

3. Endpoints:
     /getdog: Get a random image of a dog.
     /getcat: Get a random image of a cat.

Acknowledgements:
