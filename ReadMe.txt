# Pet Image API

API for fetching random images of dogs and cats.

## Overview

Pet Image API is a simple application that 
provides endpoints to retrieve random images of dogs and cats 
from external APIs.


### Prerequisites

- Node.js installed


Endpoints:
     /getdog: Get a random image of a dog.
     /getcat: Get a random image of a cat.

Acknowledgements:
     Dog images provided by Dog CEO's Dog API.
     Cat images provided by The Cat API.
